# Blood-Donation-Management-using-Blockchain
This project leverages blockchain technology to create a transparent, secure, and decentralized ecosystem for managing blood donations. By utilizing a distributed ledger, the system ensures the integrity of donor records, tracks the journey of blood units from donation to transfusion, and eliminates the risk of data tampering or centralized failures.
# Key FeaturesImmutable Donor Records:
Once a donation is recorded on the blockchain, the data cannot be altered, ensuring a permanent and reliable medical history.
- Real-time Tracking: Provides a transparent supply chain for blood banks and hospitals to track available inventory in real-time.
- Enhanced Security: Uses cryptographic principles to protect sensitive donor information while maintaining accessibility for authorized medical personnel.
- Decentralized Verification: Removes the need for a central intermediary, reducing administrative delays and increasing trust between stakeholders.
# Tech StackBlockchain:
Ethereum .Smart Contracts: Solidity.
- Front-end: ReactJS, HTML, CSS, and Bootstrap.
-  Back-end: Node.js.
- Database: SQL/MySQL (for off-chain metadata).
# How it WorksRegistration:
Donors register via the web interface, and their eligibility is verified.
- Donation Entry: When a donation occurs, a unique hash is generated and stored on the blockchain.
- Blood Testing: Results for blood grouping and safety screenings are appended to the specific unit's digital record.
- Distribution: Hospitals request blood types, and the system matches them with available, verified units in the decentralized ledger.
